
Linux commands
 
 ctrl+alt+t = команда открытия терминала
~ = символ домашней директории
/ = символ корневой директории
.. = обозначает предыдущую директорию
. = обозначает текущую директорию
 1. ls(list) = комана для вывода, отоброжения всех директорий(папка) и файлов в текущей директории в терминале
 2. cd(change directory) = команда для изменения(переходов) одной директории на другую
 cd <название директории>
 3. mkdir(make directory) = создание директории, создает указанную директорию в рабочей директории
 4. touch =  создает указанный файл
 5. rm(remove) =  удаление файла
 6. rmdir = удаление директории
 7. pwd(present working directory) = выводит путь до текущей, рабочей директории  
 8. rm -rf = принудительное удаление
 9. ls -a = выводит список как скрытых так и обычных файлов и директорий
 10. nano <path_to_file> = откроется файл для редактирования в графическом интерфейсе
 *если нет файла, то nano попытаетсяя создать новый файл
 11. cat <file_name> = выводит содержимое файла в терминал
