GIT - распределенная система контроля версий.
Это система для отслеживания и ведения истории изменения файлов, в ашем проекте.

Репозиторий - хранилище вашего кода и истории его изменений. 

GitHub - веб сайт для размещения git-репозиториев в совместной разработке проектов

Кoманды Git:
1. git init -  она создает новый локальный гит репозиторий у вас на пк  в той директории, в которой была прописана команда.(Команда вводится только один раз в начале)
2. git add - команда для добавления файлов в систему отслеживания гита, после этой команды изменения начинают отслеживаться гитом.
3. git commit -m '<comment> - как только достигаем определенного момента в разработке, то тогда мы сохраняем и комментируем изменения в локальный репозиторий.
4. git remote add - это команда для того чтобы связать наш локальный репозиторий с удаленным, то есть добавляет подключение к репозиторию на гитхабе
       git remote add <name: origin> <ссылка_на_репо:ssh>
5. git push - команда для отправки всех изменений на локальном репозитории в удаленный репозиторий
            git push origin <branch(main)>
            ---------------------------
            
            1) git init (только 1 раз)
            2) git add
            3) git commit -m ''
              3.1) git branch -m main (1 раз, для переименования ветки) 
                *) git remote add origin <ссылка> (1 раз)
            4.*) git push origin main
